First-time setup — to enable real-time sync across all devices, you need a free Firebase database. It takes 2 minutes:

  1. Go to console.firebase.google.com and sign in with Google
  2. Click "Add project" → name it (e.g., autoshop) → Continue → Continue → Create
  3. In the left menu, click "Build" → "Realtime Database" → Create Database
  4. Choose a location → Select "Start in test mode" → Enable
  5. Copy the URL shown at the top (looks like https://abc-default-rtdb.firebaseio.com)
